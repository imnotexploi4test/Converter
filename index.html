Audio/Video → 8-bit Bytebeat Converter

Runs entirely in your browser. Decodes almost any audio/video file, resamples it to mono 8-bit PCM, and produces a downloadable WAV plus a bytebeat-style JavaScript string.

Accepts virtually any audio or video container (mp3, wav, ogg, m4a, aac, flac, webm, mp4, mov, avi, mkv, wma, opus, aiff, au, amr, 3gp, flv, ts, ape, wv, etc.). Plain text/code files (.txt .html .py .js .lua .json .css .md etc.) are rejected immediately since they can never contain audio.
"Letters, digits & symbols" mode maps each byte to a 75-character ASCII run (codes 48–122: digits, symbols : ; < = > ? @, uppercase, symbols [ ] ^ _ ` ~, lowercase), then the formula recovers the sample as (d[t % d.length].charCodeAt(0) * 3) - 127 — the same technique used by other bytebeat-from-text converters. Backslash is remapped to ~ since a raw backslash cannot appear unescaped in a single-quoted string without changing the character count.
